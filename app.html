E-Ink Connect App:

With this app you can create and send images via ble to the e-ink display.

The app ist build with the ionic framework to create android and ios apps.

Used Native APIs:

  • BLE - sending images
  • Status Bar - showing the native status bar on top

Features:

  • Scanning for devices

  • Connect to device

  • Create Images:

    • Change background color
    • Change font color
    • Write text on canvas
    • Draw by hand on canvas

  • Send image to device
